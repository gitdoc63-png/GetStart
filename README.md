# GetStart
A convenient program for auto-launching on Windows
This program is for those who don't want to constantly change the startup folder
Simply enter the path, and the program will launch your path the next time it starts
The program is still under development, so I apologize for any errors or shortcomings. They will be fixed

  ↓↓↓----------HOW TO WORK WITH THE PROGRAM----------↓↓↓
  1. Run "setting" and then click on 1 to continue
  2. Click 1 and enter the path to your program, making sure that the path does not contain any letters other than English letters
  3. Repeat the first step, but this time press 2
  4. Press 1 and the program will automatically open the startup folder and transfer the "STARTPROGRAM" file there
  5. Now everything is ready, if you encounter a problem, first of all, make sure again of the path, if this does not help, you will have to create a shortcut and move the folder itself to the C drive
  ↓↓↓----------POSSIBLE ERRORS----------↓↓↓
  1. Path error you must enter the correct path or it will give you an error when you start
  2. DO NOT EVER MOVE FILES FROM ONE FOLDER TO ANOTHER OR TO YOUR DESKTOP, EXCEPT IF THE FILES HAVE THE SAME NAME AND ARE IN THE SAME FOLDER, AS THIS MAY CAUSE ERRORS, Just leave it like that, okay?
